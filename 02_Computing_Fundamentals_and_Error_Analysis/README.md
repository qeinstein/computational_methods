# 02 - Computing Fundamentals and Error Analysis

This module covers the mathematical foundations of computational methods, including how numbers are represented in computers and how errors propagate through numerical computations.

## Topics Covered

### 1. Number Representation
- Binary, decimal, and hexadecimal systems
- Floating-point representation (IEEE 754)
- Mantissa, exponent, and sign bit
- Machine epsilon and precision limitations
- Rounding and truncation errors

### 2. Error Types and Analysis
- Absolute and relative errors
- Round-off errors
- Truncation errors
- Error propagation in arithmetic operations
- Condition numbers and stability
- Error estimation techniques

### 3. Debugging Practices
- Common numerical errors and pitfalls
- Testing and validation strategies
- Debugging techniques for mathematical code
- Performance profiling
- Documentation best practices

## Files in This Module

- **01_number_representation.md**: Theory of number systems and floating-point representation
- **01_number_representation.py**: Examples of numerical representation and conversions
- **02_error_types.md**: Comprehensive error analysis and propagation
- **02_error_types.py**: Practical error calculations and demonstrations
- **03_debugging_practices.md**: Best practices for debugging numerical code
- **03_debugging_practices.py**: Debugging tools and techniques with examples

## How to Use

1. Study **01_number_representation.md** to understand computer arithmetic
2. Review **02_error_types.md** to learn error analysis
3. Learn debugging in **03_debugging_practices.md**
4. Run implementations to see concepts in action
5. Experiment with different input values to understand error behavior

## Key Learning Outcomes

By the end of this module, you should be able to:
- Explain floating-point arithmetic and its limitations
- Calculate and estimate errors in numerical computations
- Understand when and why numerical instability occurs
- Debug numerical code effectively
- Design robust and accurate numerical algorithms
- Be aware of pitfalls in computational methods

## Prerequisites

- Basic Python knowledge (from Module 01)
- Understanding of basic calculus and linear algebra
