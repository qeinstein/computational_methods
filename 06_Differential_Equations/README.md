# 06 - Differential Equations

This module covers numerical methods for solving ordinary differential equations (ODEs). These techniques are essential for modeling dynamic systems in science and engineering.

## Topics Covered

### 1. Euler Method
- Basic step-by-step integration
- Taylor series foundation
- Forward Euler method
- Error analysis and convergence
- Stability analysis
- Applications to simple ODEs

### 2. Heun's Method (Improved Euler)
- Second-order Runge-Kutta method
- Predictor-corrector approach
- Improved accuracy over Euler
- Error estimation
- Step size control

### 3. Runge-Kutta Methods
- Fourth-order Runge-Kutta (RK4)
- Higher accuracy and stability
- Multi-stage integration
- Adaptive step size methods
- System of ODEs
- Stiff equations handling

## Files in This Module

- **01_euler_method.md**: Euler method theory and derivation
- **01_euler_method.py**: Implementation and examples
- **02_heun_improved_euler.md**: Heun's method theory and improvements
- **02_heun_improved_euler.py**: Implementation with comparisons
- **03_runge_kutta.md**: Runge-Kutta methods and RK4 derivation
- **03_runge_kutta.py**: Complete RK4 implementation and applications

## How to Use

1. Start with **01_euler_method.md** for fundamental concepts
2. Learn **02_heun_improved_euler.md** for better accuracy
3. Master **03_runge_kutta.md** for practical problems
4. Run implementations on various ODEs
5. Visualize solutions and compare methods
6. Experiment with different step sizes

## Key Learning Outcomes

By the end of this module, you should be able to:
- Implement Euler, Heun, and Runge-Kutta methods
- Solve initial value problems numerically
- Understand order of accuracy and convergence
- Analyze stability of numerical methods
- Select appropriate methods for different problems
- Solve systems of ODEs
- Apply numerical ODE solving to real-world problems
