# 05 - Numerical Integration

This module covers techniques for numerically approximating integrals. Integration is vital in engineering, physics, and applied mathematics where analytical solutions are unavailable.

## Topics Covered

### 1. Trapezoidal Rule
- Basic quadrature formulas
- Trapezoid area approximation
- Error analysis and convergence
- Composite trapezoidal rule
- Practical applications

### 2. Simpson's Rule
- Parabolic interpolation
- Simpson's 1/3 rule
- Simpson's 3/8 rule
- Error comparison with trapezoidal rule
- Composite Simpson's rule
- Higher accuracy and convergence rates

## Files in This Module

- **01_trapezoidal_rule.md**: Trapezoidal rule theory and applications
- **01_trapezoidal_rule.py**: Implementation with examples and error analysis
- **02_simpsons_rule.md**: Simpson's rule theory and derivations
- **02_simpsons_rule.py**: Complete implementation and comparisons

## How to Use

1. Study **01_trapezoidal_rule.md** for fundamental concepts
2. Learn **02_simpsons_rule.md** for improved accuracy
3. Run implementations on various functions
4. Compare accuracy and computational cost
5. Visualize error behavior with different step sizes

## Key Learning Outcomes

By the end of this module, you should be able to:
- Implement trapezoidal and Simpson's rules
- Understand quadrature concepts and convergence
- Select appropriate methods for different integrands
- Estimate integration errors
- Optimize step size for desired accuracy
- Apply numerical integration to real-world problems
- Recognize when numerical integration is necessary
